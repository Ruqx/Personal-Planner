# Personal Planner App

A comprehensive personal productivity application that combines time visualization, planning, and task management into a single interactive platform.

## Features

### Circular Clock Calendar
- A unique clock-based calendar visualization  
- Time is represented using concentric circular rings (year, month, date, hour, minute, second)  
- Real-time movement of time across the rings  
- Current time is highlighted with a continuous indicator across all layers  

### Goals Integration
- Create and manage goals with title, date, and completion status  
- Goals are visually integrated into the clock calendar  
- Future goals highlight their corresponding time segments  
- Hovering over highlighted segments displays goal details  

### Notes
- Add and manage notes for quick access  
- Useful for capturing ideas, reminders, and important information  

### Kanban Board
- Organize tasks using a Kanban-style workflow  
- Drag-and-drop functionality for task movement  
- Supports multiple stages such as To Do, In Progress, and Done  

### Calendar View
- Traditional calendar view alongside the clock-based visualization  
- Easy navigation across dates and months  

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Data Handling: JSON / Local Storage / API-based storage  

## Key Highlights

- Integrates time, goals, notes, and tasks into a single system  
- Interactive and visually intuitive interface  
- Modular design allowing easy expansion  
- Focus on improving time awareness and productivity
